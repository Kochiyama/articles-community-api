# Articles Community API

This is the articles community api, articles community is an @kochiyama initiative, a place where anyone can write about anything.
Niche agnostic.

This application is developed with NestJS framework and PostgreSql Database. Both are running on a free dyno at heroku.

## API Reference

You can find the API Reference on [this swagger](https://article-community.herokuapp.com/api/#/) page
